# Personal Website

**Description**  
This is a simple personal website created to showcase skills in HTML, CSS, and JavaScript. The website includes an introductory section, an image gallery, a contact form, and simple animations to enhance user engagement.

---

## Features

- **Responsive Design**: Adjusts to various screen sizes, ensuring a consistent experience on mobile, tablet, and desktop devices.
- **Image Gallery**: A collection of five images that highlight different sections or personal interests.
- **Interactive List**: An ordered or unordered list to display content in an organized way.
- **Basic Animations**: Smooth transitions and animations to improve the visual appeal.
- **JavaScript Functionality**: A simple JavaScript function to add interactivity.
- **Contact Form**: A basic contact form allowing visitors to get in touch.
